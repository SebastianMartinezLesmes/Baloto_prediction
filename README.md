# Baloto_prediction 
### entorno virtual

1. crearlo(cmd): 
    python -m venv venv

2. activarlo: 
    venv\Scripts\Activate

3. instalar dependencias: 
    pip install -r requirements.txt

4. salir del entorno:
    deactivate

### ejecutar proyecto

1. Entrar al CMD
2. Ejecutar:
    python bot/index.py
